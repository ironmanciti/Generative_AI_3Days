# 투자 수익률(ROI) 분석 리포트 스킬

## 언제 사용하나요
사용자가 투자 원금, 현재 평가액(또는 회수 금액), 투자 기간을 제시하면서
수익률 분석이나 투자 성과 평가를 요청할 때 이 스킬을 사용합니다.

## 계산 방법
정확한 수치는 직접 암산하지 말고, 반드시 동봉된 roi_calculator.py 스크립트를
실행하여 구합니다. 모델의 암산은 오차가 생길 수 있으므로 항상 스크립트 결과를 신뢰합니다.

```
python roi_calculator.py --principal <원금> --current <현재평가액> --years <투자기간>
```

스크립트는 다음 세 가지 값을 출력합니다.
- 총수익(절대 금액)
- 총수익률(%)
- 연환산수익률 CAGR(%)

## 출력 형식
계산 결과는 반드시 아래 표준 형식으로 정리합니다.

### 투자 수익률 분석 리포트
- 투자 원금: (입력값)
- 현재 평가액: (입력값)
- 투자 기간: (입력값)년

| 지표 | 값 |
|---|---|
| 총수익 | (스크립트 결과) |
| 총수익률 | (스크립트 결과)% |
| 연환산수익률(CAGR) | (스크립트 결과)% |

### 종합 평가
연환산수익률(CAGR)을 기준으로 다음과 같이 한 줄 평가를 덧붙입니다.
- 10% 이상: 우수
- 4% 이상 10% 미만: 양호
- 0% 이상 4% 미만: 보통
- 0% 미만: 손실
